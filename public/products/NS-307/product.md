# NS-307散装遥控接收头成型机

## 产品型号

NS-307

## 设备参数

| 参数 | 内容 |
|---|---|
| 电源 | 220V AC 300W |
| 气压 | 0.4-0.6Mpa |
| 产能 | 2700pcs/H |
| 尺寸 | L1100*W600*H1500mm, |
| 重量 | 160KG |

## 产品特性

- 适用于散装遥控接收头，定制专用分方向送料盘自动上料。
- 整机触控屏界面，PLC控制气缸工作，成型流程：振动盘分方向上料 $\rightarrow$ 夹头取料 $\rightarrow$ 定位 $\rightarrow$ 剪脚 $\rightarrow$ 90度成型模 $\rightarrow$ 二次折弯模（或作为备用工位） $\rightarrow$ 退料。
- 弯脚距离 $1.5 - 10\mathrm{mm}$ 可调，脚长 $3 - 30\mathrm{mm}$ 旋钮可调，有刻度尺参照，调试方便。
- 取料夹头为仿形制作，夹头包胶，取料更稳。
- 弯脚及剪脚时配置压脚装置(先压紧引脚再成型或剪脚)，确保接收头本体不受力，弯脚时采用滚轴折弯，确保引脚表面不刮伤。
- 采用亚德客气缸及电磁阀，品质稳定寿命长，模具及刀具材料选用SKD-11，经久耐用。
- 产能屏幕设定，到数自动停机，异常卡料自动报警停机，一人可看护3-5台设备，节省人工。
