# NS-806管装三极管成型机

## 产品型号

NS-806

## 设备参数

| 参数 | 内容 |
|---|---|
| 电源 | 220V AC 60W |
| 气源 | 0.4-0.6Mpa |
| 尺寸 | L500*W450*H450mm |
| 重量 | 35KG |
| 产能 | 4000pcs/H |

## 产品特性

- 适用于管装之单一款封装的功率晶体、三极管成型剪脚，不同封装模式需要换模及轨道。
- 定制模具(模具可更换)，弯脚高度 $2.5 - 6\mathrm{mm}$ 可调，出板脚长固定(可按要求定制出板脚长可调)。
- 成型时先预压引脚再剪脚折弯，确保本体不受外力，保证加工品质。
- 外接气源同步辅助退料，确保退料顺畅，克服老款设备退料不顺问题。
