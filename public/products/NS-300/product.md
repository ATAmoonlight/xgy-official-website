# NS-300散装电容剪脚机

## 产品型号

NS-300

## 设备参数

| 参数 | 内容 |
|---|---|
| 电源 | 220V AC 90W |
| 尺寸 | L500*W400*H400mm |
| 重量 | 26KG |
| A 本体直径 | 3-20m/m |
| H 脚长 | 3-15m/m |
| W 线径 | 0.35-2.0m/m |
| 加工量 | 300PCS/Min |

## 产品特性

- 切刀结构简化设计，马达直接驱动(无需皮带或链条传动)，确保运转稳定。
- 切刀采用SKD11材质，寿命超长耐用。
- 配置动态压料装置(剪脚位置)，剪脚时压料，不剪脚时不压料，确保剪脚误差±0.2mm（物料底部平整，如电解电容）。
- 送料导轨采用激光切割加工，精度高，走料顺畅。
- 手工喂料, 适合加工本体3-20mm, 线径0.35-2.0mm, 剪脚长度3-15mm可调, 加工量1500-3000PCS/H。
- 此款设备可根据客户提供物料定制款，可选配送料盘上料节省人工。
