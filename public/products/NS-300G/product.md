# NS-300G经济型散装电容成型机

## 产品型号

NS-300G

## 设备参数

| 参数 | 内容 |
|---|---|
| 电源 | 220V AC 90W |
| 气源 | 0.4-0.6Mpa |
| 重量 | 45KG |
| 尺寸 | L650*W400*H600m/m |
| 产能 | 1500-2600pcs/H |

## 产品特性

- 适用于散装电解电容，安规电容等立式径向元件，采用平面振动送料方式，人工上料，设备自动剪脚和成型。
- 弯脚距离1.5-10mm可调，脚长3-20mm可调，脚长旋钮可调，方便调试。
- 剪脚时配置动态压料就够(先压紧本体再剪脚)，确保剪脚精度公差±0.2mm。
- 成型时先压紧引脚确保本体不受外力，确保加工品质。
- 折弯模具采用一体模, 单边即可调试尺寸, 方便实用。
