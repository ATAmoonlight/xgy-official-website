# NS-865方形物料自动穿套热缩管剪脚成型机

## 产品型号

NS-865

## 设备参数

| 参数 | 内容 |
|---|---|
| 电源 | 220V AC 1500W |
| 气源 | 0.4-0.6Mpa |
| 尺寸 | L1600*W850*H1650mm |
| 重量 | 330KG |
| 产能 | 1500-2000pcs/H |

## 产品特性

- 适用于桥堆、IGBT、三极管、安规电容等方形物料自动穿套热缩管/铁氟龙管及剪脚成型。
- 本机以人性化设计为理念，防呆模式，套管用尽和本体没有穿套管可报警停机，IO监控，人工只需做简单的放料和收料的动作。
- 采用往复连续平移工位依次完成推料、送料、送管、预压定位、预压热缩、二次热缩、切脚、成型。
- 特殊穿管模式，确保热缩后套管精度误差在正负 $0.5\mathrm{mm}$ 。
- 敞开式设计，模组式结构，方便调试及维护。
- 可成型多种型状(具体形状按贵司要求制作)。
