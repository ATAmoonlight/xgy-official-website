# NS-860编带/散装电阻保险丝穿管成型机

## 产品型号

NS-860

## 设备参数

| 参数 | 内容 |
|---|---|
| 电源 | 220V AC 1500W |
| 气源 | 0.4-0.6Mpa |
| 尺寸 | L1300*W850*H1650mm |
| 重量 | 330KG |
| 产能 | 1500-1800pcs/H |

## 产品特性

- 适用于散装/编带电阻保险丝穿热缩管/铁氟龙管，立式/卧式成型。
- 本机以人性化设计为理念，防呆模式，套管用尽和本体没有穿套管可报警停机，IO监控，人工只需做简单的放料和收料的动作。
- 采用往复连续平移工位完成定位, 整形, 穿管, 热缩, 成型等动作。
- 特殊穿管模式，确保热缩后套管精度误差在正负 $0.5\mathrm{mm}$
- 敞开式设计，模组式结构，方便调试及维护。
- 可成型多种型状(具体形状按贵司要求制作)。
