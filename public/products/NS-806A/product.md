# NS-806A自动管装气动式IGBT成型机

## 产品型号

NS-806A

## 设备参数

| 参数 | 内容 |
|---|---|
| 电源 | 220V AC 30W |
| 气源 | 0.4-0.6Mpa |
| 尺寸 | L500*W450*H550mm |
| 重量 | 40KG |
| 产能 | 2000pcs/H |

## 产品特性

- PLC控制,适合单一封装的IGBT或桥堆物料,采用气动式成型,解决传统设备成型时间不充裕的问题。
- 折弯高度3-6mm可微调,其他尺寸固定。
- 成型时先预压引脚再剪脚折弯，确保本体不受外力，保证加工品质。
